# Features

## Content Sections

| Section | Description |
|---------|-------------|
| **Last 10 Nights** | Duas and focus for the most blessed nights of Ramadan, including Laylatul Qadr |
| **Rabbana Duas** | Qur'anic supplications for guidance, mercy, forgiveness, and success |
| **Morning Adhkar** | Authentic remembrance to start the day with protection, gratitude, and trust in Allah |
| **Evening Adhkar** | Authentic remembrance to close the day with calm, protection, and repentance |

## UI Features

| Feature | Description |
|---------|-------------|
| **Theme toggle** | Switch between light and dark mode |
| **Font size control** | Adjust text size for comfortable reading |
| **Transliteration toggle** | Show or hide Arabic transliteration |
| **Copy button** | One-tap copy of duas to clipboard |
| **Share button** | Share duas with others |
| **Completion checkboxes** | Track progress through adhkar sections |
| **Sticky section index** | Quick navigation within long pages |
| **Back to top** | Return to top of page on long content |

## Authenticity Approach

- **Qur'anic duas** — With surah and ayah reference
- **Authentic hadith-based adhkar only** — From reliable sources (Sahih Bukhari, Sahih Muslim, etc.)
- **Arabic with full harakat** — Helps non-Arab readers with pronunciation
- **English and Urdu translations** — For understanding
- **Source links** — Clear references to Quran.com and Sunnah.com

## Design

- **Mobile-first** — Optimized for phones and tablets
- **Accessible** — Clear typography, contrast, and structure
- **Calm reading experience** — Minimal distractions, focused on content
