# Run Towards Allah's Mercy

A calm, mobile-first reading space for Qur'anic duas, authentic adhkar, remembrance of Allah, and spiritual reflection.

## About

This is a **Sadaqah Jariah** project built to spread beneficial remembrance, reflection, and peace. It invites readers to do taubah and run towards Allah's Mercy through authentic duas and adhkar from the Qur'an and Sunnah.

## Features

- **Qur'anic Rabbana Duas** — Powerful supplications from the Qur'an for guidance, mercy, forgiveness, and success
- **Morning and Evening Adhkar** — Authentic hadith-based remembrance for the start and end of each day
- **Last 10 Nights of Ramadan** — Special duas and focus for Laylatul Qadr
- **Arabic, English, and Urdu** — Full harakat (diacritics) in Arabic with translations
- **Theme toggle** — Light and dark mode
- **Font size control** — Adjust text size for comfortable reading
- **Copy and share** — One-tap copy and share for duas
- **Completion checkboxes** — Track progress through adhkar
- **Mobile-first design** — Optimized for reading on phones and tablets

## Tech Stack

- **Next.js** (App Router)
- **React**
- **TypeScript**
- **Tailwind CSS**

## Getting Started

### Prerequisites

- Node.js 18.x or later

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

### Build

```bash
npm run build
npm start
```

## Project Structure

| Directory | Purpose |
|-----------|---------|
| `app/` | Next.js App Router pages (home, adhkar, rabbana-duas, last-ten-nights, about, legal pages) |
| `components/` | Reusable UI components (header, footer, dua cards, adhkar pages, etc.) |
| `data/` | Content data (rabbana-duas, morning-adhkar, evening-adhkar, last-ten-nights) |
| `lib/` | Utilities, types, content config, and source links |
